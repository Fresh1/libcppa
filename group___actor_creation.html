Spawns a new context-switching or thread-mapped actor that executes fun(args...) and joins grp immediately.

Parameters:
funA function implementing the actor's behavior.
argsOptional function parameters for fun.
Template Parameters:
HintA hint to the scheduler for the best scheduling strategy.
Returns:
An actor_ptr to the spawned actor.
Note:
The spawned actor joins grp after its init member function is called but before it is executed. Hence, the spawned actor already joined the group before this function returns.
