Detailed Description

This is the root namespace of libcppa.

Thie cppa namespace contains all functions and classes to implement Actor based applications.

Denotes whether a user wants an actor to take part in cooperative scheduling or not.

Enumerator:

Indicates that an actor should run in its own thread.
