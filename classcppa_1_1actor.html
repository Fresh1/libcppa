virtual bool cppa::actor::attach ( attachable *  ptr)
pure virtual

Attaches ptr to this actor.

The actor will call ptr->detach(...) on exit, or immediately if it already finished execution.

Parameters:
ptrA callback object that's executed if the actor finishes execution.
Returns:
true if ptr was successfully attached to the actor; otherwise (actor already exited) false.
Warning:
The actor takes ownership of ptr.
