static intrusive_ptr<group> cppa::group::anonymous ( )
static

Returns an anonymous group.

Each calls to this member function returns a new instances of an anonymous group. Anonymous groups can be used whenever a set of actors wants to communicate over an exclusive channel.

Examples:
dining_philosophers.cpp.
