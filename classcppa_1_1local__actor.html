virtual void cppa::local_actor::dequeue ( partial_function &  pfun)
pure virtual

Removes the first element from the mailbox pfun is defined for and invokes pfun with the removed element. Blocks until a matching message arrives if pfun is not defined for any message in the actor's mailbox.

Parameters
pfunA partial function denoting the actor's response to the next incoming message.
Warning
You should not call this member function by hand. Use the receive function or the become member function in case of event-based actors.

virtual void cppa::local_actor::exec_behavior_stack ( )
virtual

Executes an actor's behavior stack until it is empty.

This member function allows thread-based and context-switching actors to make use of the nonblocking behavior API of libcppa. Calling this member function in an event-based actor does nothing.
