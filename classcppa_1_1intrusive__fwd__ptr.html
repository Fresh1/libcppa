template<typename T, typename Ref, typename Deref>
class cppa::intrusive_fwd_ptr< T, Ref, Deref >

A reference counting smart pointer implementation that can be used with forward declarated types.

libcppa assumes T is derived from ref_counted and implicitly converts intrusive_fwd_ptr to intrusive_ptr
