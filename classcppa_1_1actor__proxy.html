any_tuple  msg

pure virtual

Delivers given message via this proxy instance.

This function is meant to give the proxy the opportunity to keep track of synchronous communication or perform other bookkeeping if needed. The member function is called by the protocol from inside the middleman's thread.

Note
This function is guaranteed to be called non-concurrently.
